Creating an upload file

Before you start

You will need to know the agreement ID you have with each employer, this can be found on your Organisation and agreements page.

The CSV file is limited to 100 rows.

Uploading a CSV file

You can:

  • use your learner management system (LMS) to create a CSV file to upload 1 or more apprentices for 1 or more employers at the same time. If you don't use a LMS then you can use our template
  • use existing cohort references to add new apprentices
  • auto-generate new cohorts by leaving the cohort field empty
  • check for errors
  • approve apprentice details and send to more than 1 employer at the same time
