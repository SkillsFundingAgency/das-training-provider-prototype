Upload a CSV file

Your learner management system (LMS) may be able to create a CSV file for you or you can use our template to create one.

There is a full specification on uploading new apprentices using a CSV file on GOV.UK.

You can add new apprentices to an existing cohort if you already have one, or we will create a new one for each employer.

Help with finding Agreement ID
You can find the Agreement ID you have with an employer on your Organisation and agreements page.

Once uploaded, you'll be able to:

  • check for errors
  • correct records and upload again
  • save and send to employer(s) to approve
