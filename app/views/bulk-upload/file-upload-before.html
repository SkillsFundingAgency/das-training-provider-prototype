Using file upload

Before you start

There is a full specification on uploading new apprentices using a CSV file on GOV.UK.

You will need to know the agreement ID you have with each employer, this can be found on your Organisation and agreements page.

The CSV file is limited to 100 rows.

How it works

  • use file upload to add 1 or more apprentices for 1 or more employers
  • if you have permission from the employer you can create new cohorts by leaving the 'CohortRef' field empty; the cohort will be created when you save or approve the file
  • add apprentices to existing cohorts that have not yet be approved by the employer
  • check for errors

Creating a CSV file

You can:

  • use your learner management system (LMS) to create a CSV file
  • use our template if you do not use a LMS
