Using an upload file

Before you start

You will need to know the agreement ID you have with the employer; this can be found on your Organisation and agreements page.

Upload a CSV file

You can:

  • use your learning management system (LMS) to create a CSV file to upload 1 or more apprentices for 1 or more employers at the same time. If you don't use a LMS then you can use our template
  • use existing cohorts or auto-generate new ones
  • check for errors
  • amend errors in your data file
  • approve apprentice details and send to more than 1 employer at the same time
