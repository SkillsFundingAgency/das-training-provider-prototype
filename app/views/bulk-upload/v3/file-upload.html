Upload a CSV file

You must use a CSV file; if you have a management information system, it may be able to create a CSV file for you.

You can also use the apprenticeship service bulk upload template.

Uploading a file means you can add 1 or more apprentices from 1 or more employers at the same time. You can use an existing cohort or they can be created automatically.

Once uploaded, you'll be able to:

  • check for errors
  • correct records and upload again
  • save and send to employer(s) to approve
! Warning Uploading a file will overwrite any existing apprentices. Only new apprentices can be added.
